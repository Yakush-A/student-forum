# student forum

Проект представляет собой форум аналогично StackOwerflow.

## Сущности
1. Post - публикация на форуме
2. User - пользователь форума
3. Response - ответ на вопрос форума
4. Article - статья
5. Thread

## Checkstyle
В проекте используется sun_checks с некоторыми изменениями

## Сборка и запуск
### Сборка
```./gradlew clean build```
### Запуск
```./gradlew bootRun```
