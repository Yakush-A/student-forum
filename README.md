# Student forum

Проект представляет собой форум аналогично StackOwerflow/Хабр.

## Сущности
1. **Post** - публикация на форуме (вопрос пользователя). Содержит текст сообщения, данные автора, дату публикации;
2. **User** - пользователь форума. Данные пользователя - имя, тег, собственные публикации;  
3. **Response** - ответ на вопрос форума. Содержит оригинальную публикацию, текст ответа, и данные аналогично Post;
4. **Article** - статья на форуме. Имеет возможность продвижения, данные аналогично Post;
5. **Thread** - тема, содержит вопросы и статьи по конкретной теме.
